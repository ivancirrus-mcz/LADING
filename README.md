# FESTAPRINT - Landing Page

Esta é a landing page oficial do sistema FESTAPRINT, voltado para automação de artes e impressão para papelaria personalizada focada em marketplaces.

## Estrutura do Projeto

- `index.html`: Estrutura principal da página.
- `style.css`: Design system e estilos gerais.
- `logo.css`: Estilos específicos da logomarca animada.
- `Dockerfile`: Configuração para deploy via Docker (Nginx).

## Deploy no Easypanel

1. Crie um novo serviço do tipo **App**.
2. Aponte para este repositório no GitHub.
3. O Easypanel detectará o Dockerfile e fará o build automaticamente.
